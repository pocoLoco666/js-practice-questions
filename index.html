Variables and Data Types

  1. Easy:

    1. What is a variable in JavaScript?
    2. Declare a variable called age and assign the value 25 to it.
    3. What are the three primary data types in JavaScript?
    4. How do you declare a constant variable in JavaScript?
    5. Convert the string "42" to a number.

  2. Medium:

    1. Explain the difference between let, const, and var.
    2. What is the value of typeof "Hello"?
    3. Create an array with three elements: "apple", "banana", and "cherry".
    4. Declare an object called person with properties name and age.
    5. Write a code snippet that swaps the values of two variables, x and y.

  3. Hard:

    1. What is variable hoisting in JavaScript, and how does it work?

    3. Explain the concept of pass-by-reference and pass-by-value in JavaScript.
    4. Create a deep clone of a complex object without using external libraries.
    5. Write a function that determines if a variable is an array.

Conditional Statements

  1. Easy:

    1. Write an if statement to check if a number is greater than 10 and display a message.
    2. Use if and else to determine if a user is eligible to vote based on their age.
    3. Check if a variable is defined and not null before using it.

  2. Medium:

    1. Create a program that determines if a year is a leap year or not using if...else if...else.

    2. Implement a function that sorts an array of numbers in ascending order using conditional statements.

Loops

  1. Easy:

    1. Use a for loop to display numbers from 1 to 5 in the console.
    2. Write a while loop that counts down from 10 to 1 and displays each number.
    3. Use a for...in loop to iterate through the properties of an object.

  2. Medium:

    1. Create a program that calculates the sum of all even numbers from 1 to 50 using a loop.
    2. Implement a function that finds the first occurrence of a given element in an array.
    3. Write a program that generates the Fibonacci sequence up to a specified number of terms using a loop.

  3. Hard:

    1. Implement a custom forEach function for arrays that iterates through the elements and performs an action.
